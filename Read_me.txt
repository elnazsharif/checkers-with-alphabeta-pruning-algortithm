You can run this game with Visual Studio Code software.
open the whole folder in the software and go to the main.py file.
Then, from the top menu of software go to the Run tab and click the button of Run without debugging.

1.Before running at first you should install python in your computer.
This is the guidance of how to install python
https://www.digitalocean.com/community/tutorials/install-python-windows-10

2. Second, you should install pip in your computer. 
This is the guidance of how to install pip
https://www.dataquest.io/blog/install-pip-windows/

3.Then you should install pygame framework in your computer.
This is the guidance of how to install pygame
https://www.pygame.org/wiki/GettingStarted

After installing these, you are ready to run the game.
